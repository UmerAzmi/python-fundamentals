# 🌐 Streamlit (Web App Development)

This folder contains my learning and practical implementations using **Streamlit**, a Python library for building interactive web applications quickly and easily.

Streamlit is commonly used for:

- Building data apps
- Creating dashboards
- Rapid prototyping
- Deploying ML and data projects

---

## 📂 Files Included

### 1.py – Streamlit UI Components Overview

Covers basic Streamlit features including:

- Titles, headers, and markdown
- Text and code display
- User input widgets (text input, slider, multiselect, checkbox, radio)
- File uploader
- Buttons and animations
- Status messages (success, error, warning, info)
- Sidebar navigation
- Layout using columns
- Expanders
- Progress bar simulation

### 2.py – Interactive Widgets & Layout Demo

Expands on UI functionality with:

- Dynamic user input handling
- Selectbox and multiselect examples
- Sidebar-based navigation system
- Conditional rendering
- Column layouts (3-column structure)
- Expandable content sections
- Progress bar with time delay simulation

This file focuses on interaction flow and layout organization. :contentReference[oaicite:1]{index=1}

### 3.py – Rock, Paper, Scissors Game (Mini Project)

A fully functional interactive game built with Streamlit.

Features:

- User vs Computer gameplay
- Randomized computer choice
- Game logic handling
- Score tracking using `st.session_state`
- Live scoreboard using `st.metric`
- Reset functionality

This project demonstrates:

- State management
- Conditional logic
- Interactive UI updates
- Real-time score tracking
