# 🖼️ PIL / Pillow (Image Processing)

This folder contains my learning and practice work using **PIL (Pillow)**, a Python library for opening, manipulating, and saving image files.

Pillow is commonly used for:

- Opening and displaying images
- Image resizing and cropping
- Image format conversion
- Basic image editing operations
- Integrating image handling in Python applications

---

## 📘 File Included

### 1.ipynb

This notebook covers:

- Opening and displaying images
- Checking image properties (size, mode, format)
- Resizing images
- Cropping images
- Rotating images
- Saving modified images
- Basic image manipulation techniques

The notebook is structured step-by-step to understand how image processing works in Python.
