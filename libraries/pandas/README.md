# 📊 Pandas (Data Analysis & Manipulation)

This folder contains my learning and practice work using **Pandas**, a powerful Python library for data analysis and manipulation.

Pandas is commonly used for:

- Working with structured data
- Data cleaning and preprocessing
- Data analysis
- Handling CSV and Excel files
- Performing operations on rows and columns
