# 📸 OpenCV (Computer Vision)

This folder contains my learning and implementation of **OpenCV in Python** for basic computer vision tasks.

OpenCV (Open Source Computer Vision Library) is used for:

- Image processing
- Video processing
- Object detection
- Computer vision applications

---

## 📘 File Included

### OpenCV_Basics.ipynb

This notebook covers:

- Reading and displaying images
- Image resizing
- Converting images to grayscale
- Basic image transformations
- Working with image windows
- Saving processed images

The notebook is written in a step-by-step format for learning and experimentation.
