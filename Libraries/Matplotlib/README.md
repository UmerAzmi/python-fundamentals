# 📈 Matplotlib (Data Visualization)

This folder contains my learning and practice work using **Matplotlib**, a Python library used for creating visualizations and graphical representations of data.

Matplotlib is commonly used for:

- Plotting graphs
- Visualizing trends and patterns
- Comparing datasets visually
- Creating charts for analysis and reporting

---

## 📘 File Included

### Matplotlib_Basics.ipynb

This notebook covers:

- Creating line plots
- Plotting multiple lines on a graph
- Adding titles and labels
- Customizing axes
- Adding legends
- Creating bar charts
- Creating basic visual comparisons

The notebook focuses on understanding how to generate and customize visualizations step-by-step.
