# 📚 Python Libraries

This folder contains my structured learning and practical implementations of various **Python libraries** used in data analysis, visualization, computer vision, web apps, and AI workflows.

Each library is organized in its own dedicated folder.  
Every folder contains its **own README file** that explains the concepts covered and provides detailed information about the included files.

The goal of this section is to:
- Understand how different libraries work
- Learn how they integrate together
- Build practical, project-oriented knowledge

---

## 📂 Libraries Included

### 🔢 NumPy
Foundation for numerical computing.
- Array creation and memory understanding
- Indexing and slicing
- Mathematical and statistical operations
- File handling
- Real-world usage in AI workflows

---

### 📊 Pandas
Data analysis and manipulation.
- Series and DataFrames
- Data inspection and filtering
- Sorting and basic cleaning
- Working with structured datasets

---

### 📈 Matplotlib
Data visualization.
- Line plots and bar charts
- Labels and legends
- Graph customization
- Visual comparison of datasets

---

### 🖼️ PIL / Pillow
Image processing fundamentals.
- Opening and displaying images
- Resizing and cropping
- Basic image manipulation

---

### 📸 OpenCV
Computer vision basics.
- Image reading and display
- Grayscale conversion
- Basic image transformations

---

### 🌐 Streamlit
Web app development with Python.
- UI components and widgets
- Layout management
- Sidebar navigation
- State management
- Interactive mini project implementation

---

## 🎯 Purpose of This Folder

- Build hands-on understanding of important Python libraries

- Prepare for real-world projects involving:
  - Data analysis
  - Visualization
  - Web apps
