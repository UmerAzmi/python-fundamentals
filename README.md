## 📂 Repository Structure

### 📘 core/
Contains fundamental Python concepts.

- Core topics written in both `.py` and `.ipynb` formats
- Covers basics → functions → OOP → advanced topics
- Structured progressively for strong foundational understanding

Each file focuses on clear explanations and hands-on examples.

---

### 🧠 dsa/
Contains Data Structures and Algorithms practice.

- Basic array and string problems
- Pattern problems
- Element manipulation (max, second largest, duplicates, rotation, etc.)
- Jupyter notebooks for step-by-step logic building
